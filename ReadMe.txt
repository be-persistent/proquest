Setup instructions

1) Install Java
2) Install Eclipse
3) Download Selenium WebDriver package
4) Unzip Selenium to project directory (or where desired)
5) In Eclipse, create a new Java project
6) Right-click on "newproject" and select Properties.
7) On the Properties dialog, click on "Java Build Path"
8) Click on the Libraries tab
9) Click on "Add External JARs.."
10) Point to step #4

Develop methods
1) Create method to write all headers from Google search result = ProQuest
2) Create method to take a screenshot of ProQuest page search = QA

All files are written to the project root
